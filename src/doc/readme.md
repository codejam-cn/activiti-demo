# activiti 入门操作
**创建一个activiti工作流，主要包括以下几步**
- 定义流程
> 按照BPMN规范，使用流程定义工具，用流程符号把整个流程描述出来
-  部署流程
> 把画好的流程定义文件，加载到数据库中，生成表的数据
-  启动流程
> 使用java代码来操作数据库表中的内容
